Как поменять DNS сервер?

РусскийRu
  • English En

Шаг 1. Откройте настройки

Для этого нажмите на шестеренку в правом верхнем углу основного окна программы

Шаг 2. Откройте настройки сети

Для этого нажмите на "Настройки сети" (Network settings)

Шаг 3. Укажите нужный DNS сервер

Необходимо указать только IP-адреса - запросы идут через ваш VPN туннель по стандартному DNS протоколу.

Начиная с версии 2.0.8.2 в AmneziaVPN изменена реализация контейнера с DNS сервером - он теперь только для внутреннего использования (не будет доступен извне). Использоваться он будет только если его (DNS контейнер) установить вручную и поставить галочку в настройках сети (она уже стоит по умолчанию).

Замечание: если в браузере Вы указали использование DNS-over-HTTPS, то DNS запросы из этого браузера в итоге пойдут именно на указанный там сервер, независимо от настроек DNS в AmneziaVPN! Это же касается и любых других приложений, которые делают DNS запросы через нестандартные протоколы (DNS-over-HTTPS, DNS-over-TLS, DNSCrypt, DNSSEC и т.д.), поэтому рекомендуется проверить настройки своих браузеров и прочих приложений.
