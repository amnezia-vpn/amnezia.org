Для чего этот список?

AmneziaVPN довольно молодой проект и развивается не так быстро, как хотелось бы. Основной упор сейчас идет на ПК версию для платформ Windows и MacOS. На момент релиза 2.0.8 имеются:
  • версия под Windows (32-битная и 64-битная для архитектуры x86)
  • версия под MacOS (64-битная для архитектуры x86)
  • версия под Android
Востребованных устройств, очевидно, больше, но невозможно сделать для всех сразу. Поэтому ниже Вы сможете найти клиенты для различных платформ для протоколов, настраиваемых в AmneziaVPN.

Клиенты для "OpenVPN"

OpenVPN Connect - официальный клиент для данного протокола под все основные платформы.

  • Версия для Windows - подойдет для Windows 7 и новее. Поддержка ARM архитектуры не уточняется
  • Версия для Linux - в данном случае OpenVPN предоставляет ссылку на актуальную инструкцию по установке и использованию посредством терминала
  • Версия для MacOS - для версии Yosemite (10.10) и новее. Поддержка ARM архитектуры (процессоров Apple M1) не уточняется
  • Версия для Android - для Android 4.4 и новее
  • Версия для iOS - для iOS и iPadOS 9 и новее

OpenVPN for Android - сторонний клиент, предоставляющий доступ к большему количеству настроек и поддерживающий чуть более старую версию - Android 4.1 и выше.

  • Версия с Google Play - подойдет для большинства
  • Версия с F-droid - подойдет тем, у кого нет сервисов Google на устройстве
  • Версия с Github - подойдет тем, кто предпочитает просто скачивать apk-файлы

Некоторые роутеры тоже поддерживают возможность настройки OpenVPN соединения

Клиенты для "OpenVPN over Shadowsocks"

Shadowsocks - официальные клиенты под Windows, MacOS и Android + ChromeOS

  • Версия для Windows - подойдет для Windows 7 с актуальными обновлениями безопасности, для Windows 8.1, для Windows 10 версии 1607 и новее, а так же для Windows Server 2012 и новее
  • Версия для MacOS - для версии Sierra (10.12) и новее
  • Версии для Android:
    • Версия для Android и ChromeOS - согласно Google Play - "требуемая версия Android зависит от устройства"
    • Версия для Android TV - подойдет для Android 6 и новее
    • Релизы на Github - для тех кто без Google сервисов

ShadowRocket - сторонний Shadowsocks клиент для iPhone и iPad, единственным минусом которого является стоимость в 279 рублей

  • Версия для iOS - для iOS и iPadOS 9 и новее

Клиенты для "OpenVPN over Cloak"

Cloak - плагин для обфускации траффика и использования вместе с другими протоколами. AmneziaVPN использует OpenVPN over Cloak для ПК, но так же содержит в контейнере Shadowsocks для обеспечения возможности подключиться с мобильных устройств. Официальным каналом распространения плагина являются релизы на Github (рекомендуется использовать функцию поиска по странице):

  • Для ПК платформ
    • Для Windows: искать "ck-client-windows", обычно есть два файла с пометками "386" (для 32-битной версии) и "amd64" (для 64 битной версии)
    • Для Linux: искать "ck-client-linux", на момент составления списка имеется 10 файлов (по 32-битной и 64-битной версии) для архитектур x86, arm, mips, mipsle, mips_softfloat
    • Для MacOS: искать "ck-client-darwin", на момент составления списка есть сборка только для "amd64" (64-битных процессоров с архитектурой x86, не для новых процессоров Apple M1)
  • Для Android - предназначен для использования в официальном Shadowsocks клиенте
ПК-версии плагина в теории могут использоваться со сторонним ПО, однако для этого варианта нет каких-то заранее заготовленных вариантов. Таким образом, если Вы хотите расшарить контейнер с Cloak - используйте либо AmneziaVPN, либо Shadowsocks вместе с плагином.

Упомянутый выше ShadowRocket для iOS поставляется с некоторым набором плагинов "из коробки", среди которых есть в том числе и Cloak

Клиенты для "Wireguard"

Wireguard - официальные клиенты под большинство платформ

  • Версия для Windows - подходит для Windows 7 и новее, поддерживает ARM архитектуру
  • Версии для Linux - обширный список поддерживаемых дистрибутивов идет в списке после Debian (по какой то причине Ubuntu поместили отдельно выше)
  • Версия для MacOS - подходит для версии Mojave (10.14) и новее
  • Версия для Android и она же с F-droid - для Android 5.0 и новее. Помимо прочего упоминается Termux, являющийся, вероятно, самым продвинутым терминалом на Android с обширной пакетной базой
  • Версия для iOS - для iOS и iPadOS 12 и новее
  • Так же, на странице со списком загрузок для различных ОС есть упоминание OpenWRT (прошивка для роутеров)

Клиенты для "IPSec"

AmneziaVPN в контейнере с IPSec использует IKEv2 и предоставляет экспорт конфигов для:
  • strongSwan - кроссплатформенный клиент для IPSec IKEv2 для Linux, FreeBSD, MacOS и Android. Примечание - на текущий момент не рекомендуется к использованию (требуются багфиксы экспорта)
    • Для Linux и FreeBSD - сообщается о том, что такие дистрибутивы как Arch, Debian, Ubuntu, openSUSE, Gentoo, Fedora, RHEL и CentOS имеют пакеты со strongSwan в репозиториях
    • Для MacOS - доступна установка из Homebrew
    • Для Android - для Android 4.0.3 и новее
  • Устройства Apple - экспортируется файл формата plist
  • Сертификат p12 - большинство современных устройств (кроме Android) имеет встроенную поддержку IKEv2, и имеют возможность настройки подключения "из коробки". Для подключения на них требуется знать IP-адрес сервера и иметь p12 сертификат
