Подключение через файл

РусскийRu
  • English En

Подключиться через файл конфигурацией вы сможете на первой странице приложения при запуске.

Ваш файл должен иметь формат .config

Мы не рекомендуем пользоваться файлами конфигурации из публичных источников, так как они могут быть размещены с целью перехвата ваших личных данных.

Добавление сервера через файл конфигурации после первой установки.

Если вы уже установили сервер, и хотите добавить еще один, вы можете сделать это двумя способами:
Кликните на сервер, откроется экран со всеми вашими установленными серверами
Или кликните на значок настроек в правом верхнем углу экрана, а затем выберете серверы “Servers”
Кликните на знак “+” чтобы добавить сервер
Загрузите файл, для этого кликните “open file” и выберете файл. Файл должен быть в формате .config
Готово!
