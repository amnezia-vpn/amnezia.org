Setting up a server with its own DNS

  • Русский Ru

Connect with the Amnezia client to your VPN.

If you do not have a VPN, then create one using the Run Setup Wizard иor using Configure VPN protocols manually.

Then you can proceed in two ways:

Click on “Proto”l. You will see a screen with your protocols.
Or click the gear icon on the top right corner of the screen, and then select “Server Settings”.
In the server settings menu, select “Protocols and Services”
Further, in both cases, you need to click on “Install new protocols container” and on the next screen select “DNS Service”
You will see options: the port used (it cannot be changed here) and the ability to select the TCP / UDP protocol. You can also choose not to make any changes and simply click "Continue"
After selecting the container, you will be able to choose the TCP or UDP network protocol if the selected VPN protocol supports it.
For example, OpenVPN can work with both TCP and UDP.
WireGuard only works with UDP.
In the case of OpenVPN over ShadowSocks your traffic will be routed through ShadowSocks, which operates exclusively over TCP. You can also choose not to make any changes and continue with the installation.
