Removing AmnesiaVPN from the server

EnglishEn
  • Русский Ru

You have two options to choose from for your actions:

Click on “Protocol”. You will see a screen with your protocols.
Or click the gear icon on the top right corner of the screen, and then select “Server Setting”

Next, in both cases, you will see the “Server setting” window

Click on “Clean server from Amnesia software” during the removal process, all buttons will become inactive. After the process is complete, all buttons will become active again, and you will see the message "Amnesia server successfully uninstalled" at the bottom.
Next, you can manually add a new server by clicking on the “Protocols and Services” button or remove the server in the AmnesiaVPN client by clicking on the “Forget this server” button.
Done!
