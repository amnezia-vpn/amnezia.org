Adding a server via connection code after installation

If you have already installed a server and want to add another one, you can do it in two ways:
Click on a server and a screen will open with all your installed servers.
Or click on the settings icon in the upper right corner, and then select “Servers”.
To add a server, Click on the “+” sign.
Paste the Connection code into the “Connection code” line.
Done!
