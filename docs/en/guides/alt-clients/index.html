What is the purpose of this list?

You don't have to use a VPN specifically created by Amnezia with the Amnezia client. Below you can find alternative clients for various platforms and protocols.

Clients for "OpenVPN"

OpenVPN Connect - is the official client for the OpenVPN protocol, available for major platforms.

  • Windows version - Compatible with Windows 7 and newer. Support for ARM architecture is not specified.
  • Linux version - OpenVPN provides a link to the up-to-date installation and usage instructions via the terminal.
  • MacOS version - Compatible with Yosemite (10.10) and newer. Support for ARM architecture (Apple M1 processors) is not specified.
  • Android version - Compatible with Android 4.4 and newer.
  • iOS version - Compatible with iOS and iPadOS 9 and newer.

OpenVPN for Android - A third-party client that offers access to more settings and supports a slightly older version, Android 4.1 and above.

  • Google Play version - Suitable for most cases.
  • F-droid version - Suitable for those who do not have Google services on their device.
  • Github version - Suitable for those who prefer to directly download APK files.

Additionally, some routers also support the configuration of OpenVPN connections.

Clients for "OpenVPN over Shadowsocks"

Shadowsocks - official clients for Windows, MacOS, and Android + ChromeOS.

  • Windows version - Suitable for Windows 7 with the latest security updates, Windows 8.1, Windows 10 version 1607 and newer, as well as Windows Server 2012 and newer.
  • MacOS version - Compatible with Sierra (10.12) and newer. новее
  • Android versions:
    • Android and ChromeOS version - According to Google Play, the required Android version depends on the device.
    • Android TV version - Compatible with Android 6 and newer.
    • GitHub releases - For those who do not use Google services.

ShadowRocket - A third-party Shadowsocks client for iPhone and iPad, with the only drawback being its cost of 3$.

  • iOS version - Compatible with iOS and iPadOS 9 and newer.

Clients for "OpenVPN over Cloak"

Cloak - is a plugin used for traffic obfuscation and can be used alongside other protocols. AmneziaVPN utilizes OpenVPN over Cloak for PC but also includes Shadowsocks in its container for the ability to use it through a third-party Shadowsocks client. The official distribution channel for the plugin is the GitHub releases (it is recommended to use the page search function).

  • For PC platforms:
    • For Windows: Search for "ck-client-windows," typically there are two files labeled "386" (for 32-bit version) and "amd64" (for 64-bit version).
    • For Linux: Search for "ck-client-linux." At the time of list compilation, there are 10 files (for 32-bit and 64-bit versions) for x86, arm, mips, mipsle, mips_softfloat architectures.
    • For MacOS: Search for "ck-client-darwin." At the time of list compilation, there is only a build available for "amd64" (64-bit processors with x86 architecture, not for newer Apple M1 processors).
  • For Android - It is intended for use with the official Shadowsocks client.
PC versions of the plugin can theoretically be used with third-party software, but there are no preconfigured options for this scenario. Therefore, if you want to share the Cloak container, use either AmneziaVPN or Shadowsocks together with the plugin.

The aforementioned ShadowRocket for iOS comes with a set of pre-installed plugins, including Cloak.

Clients for "Wireguard"

Wireguard - official clients available for most platforms.

  • Windows version - Compatible with Windows 7 and newer, supports ARM architecture.
  • Linux version - There is an extensive list of supported distributions, with Debian mentioned separately (for some reason, Ubuntu is listed above it).
  • MacOS version - Compatible with Mojave (10.14) and newer.
  • Android version и (also available on F-droid) - Compatible with Android 5.0 and newer. It also mentions Termux, which is likely the most advanced terminal emulator for Android with an extensive package base.
  • iOS version - Compatible with iOS and iPadOS 12 and newer.
  • Additionally, on the download page for various operating systems, there is a mention of OpenWRT (firmware for routers).

Clients for "IPSec" (IKEv2)

AmneziaVPN, using IPSec in its container, utilizes IKEv2 and provides config exports for:
  • strongSwan - A cross-platform client for IPSec IKEv2 available for Linux, FreeBSD, MacOS, and Android. Note: It is currently not recommended for use due to required export bug fixes.
    • For Linux and FreeBSD - It is mentioned that distributions such as Arch, Debian, Ubuntu, openSUSE, Gentoo, Fedora, RHEL, and CentOS have strongSwan packages in their repositories.
    • For MacOS - Installation is available through Homebrew.
    • For Android - Compatible with Android 4.0.3 and newer.
  • Apple devices - Exported in plist file format.
  • p12 certificate - Most modern devices (excluding Android) have built-in support for IKEv2 and can be configured for connection "out of the box." To connect on such devices, you need to know the server's IP address and have a p12 сертификат.
