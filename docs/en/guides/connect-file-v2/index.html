You can connect through the configuration file on the first application page on startup.

Your file must be in .config format

We do not recommend using links from public sources, as they may be placed in order to intercept your personal data.

Adding a server through a configuration file after the first installation.

If you have already installed a server and want to add another one, you can do it in two ways:
Click on a "Server" and a screen will open with all your installed servers
Or click on the settings icon in the upper right corner of the screen, and then select “Servers”
To add a server, click on the “+” sign.
Upload the file, to do this, click “open file” and select the file. The file must be in .config format.
